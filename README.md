# 🛒 Retail Store Management SQL Project

## 📌 Project Overview

This project is a comprehensive SQL portfolio project based on a Retail Store Management System. It demonstrates SQL concepts ranging from basic queries to advanced analytical queries using a realistic retail database.

The project is designed to showcase SQL skills commonly required for Data Analyst and Business Analyst roles.

---

## 🗂️ Database Overview

The database consists of **9 interconnected tables**:

- Employees
- Customers
- Products
- Categories
- Orders
- Order_Items
- Stores
- Suppliers
- Promotions

These tables simulate the operations of a retail business, including sales, inventory, employees, customers, and promotional campaigns.
